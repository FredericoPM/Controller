# Controller
 Controller Ã© um projeto de IOT que consiste em um aplicativo mobile feito em flutter que se comunica via Wi-fi utilizando-se de MQTT com um micro controlador. Assim possibilitando o usuario controlar diversos aparelhos a distancias, como luzes (normais ou rgb), motores, utensilhos domesticos, etc.. Nesse repositorio temos todos os codigos fontes do App controller e da sua connectividade com o ESP-32 ou qualquer outro micro controlador com acesso ao wifi, alem de um script em python que possibilita o armazenamento de todas as leituras e envios em um banco de dados local.
